### [Volta](https://www.volta.sh) for controlling dev environment

- Using to control dev environment in a team
- Config in `package.json` file
  ```json
  {
    ...
  ,
  "volta": {
    "node": "20.13.1"
  },
    ...
  }
  ```
- Sync dev environment only in the project, not affecting global environment
